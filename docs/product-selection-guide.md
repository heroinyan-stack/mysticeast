## 五、产品定价策略

### 定价公式

```
建议售价 = 成本价 × 2.5

示例：
黑曜石手串：$3 × 2.5 = $7.5 → $29（心理定价）
紫水晶手串：$4 × 2.5 = $10 → $39
黄水晶手串：$4 × 2.5 = $10 → $35
水晶树摆件：$5 × 2.5 = $12.5 → $49
桃木符咒：$2 × 2.5 = $5 → $19

原价设置 = 建议售价 × 1.3
折扣显示 = (1 - 建议售价/原价) × 100%
```

### 价格心理学

```
✅ $29（比$30感觉便宜）
✅ $39（比$40感觉便宜）
✅ $49（比$50感觉便宜）

❌ $27（非整数，显得不专业）
❌ $33.33（过于精确）
❌ $50（整数，显得贵）
```
