## 一、风水吉祥摆件类（高利润，差异化强）

### 1. 纯铜貔貅摆件（Brass Pixiu Figurine）
```
🔍 CJ搜索词：brass pixiu, feng shui pixiu, pi yao
💰 成本：$4-12（看大小）
💰 建议售价：$49.99-79.99
📈 为什么好卖：
  - 貔貅是中国最有名的招财神兽，老外也认
  - 纯铜材质，有质感，利润空间大
  - 风水爱好者愿意花钱
💡 选品要点：
  - 选黄铜材质的，不要树脂
  - 一对卖（公母一对）比单个好
  - 配红色底座和说明卡
```

### 2. 五帝铜钱挂饰（Five Emperor Coins）
```
🔍 CJ搜索词：five emperor coins, chinese feng shui coins, 5 coins string
💰 成本：$1-3
💰 建议售价：$19.99-24.99
📈 为什么好卖：
  - 客单价低，冲动消费
  - 多场景：挂门上、车里、钱包里
  - 可以和其他产品搭配卖
💡 选品要点：
  - 红绳编织款最经典
  - 配中国结更有东方味
  - 一定要附英文说明卡（讲清楚五帝是哪五帝、怎么用）
```

### 3. 八卦镜（Bagua Mirror）
```
🔍 CJ搜索词：bagua mirror, feng shui mirror, pakua mirror
💰 成本：$2-5
💰 建议售价：$24.99-34.99
📈 为什么好卖：
  - 风水必备品，认知度高
  - 凹镜/凸镜两款，功能不同
  - 挂门口，化煞挡煞
💡 选品要点：
  - 有凸镜（挡煞）和凹镜（聚气）两款
  - 桃木边框比塑料好
  - 直径 10-15cm 最合适
```

### 4. 铜葫芦摆件（Brass Wu Lou / Gourd）

💡 选品要点：
  - 黄铜材质，带八卦图案的最好
  - 带底座的摆件款
  - 可以开盖的款（放符咒/硬币）
```

### 5. 金蟾摆件（Three-Legged Toad / Money Frog）

💰 成本：$3-8

💡 选品要点：

### 6. 风水罗盘（Feng Shui Compass / Luopan）
```
🔍 CJ搜索词：feng shui compass, luopan, bagua compass
💰 成本：$6-15
💰 建议售价：$49.99-79.99
